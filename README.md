# Sistem-Monitoring-Progress-Konstruksi
SIMPROK (Sistem Monitoring Progres Konstruksi) adalah sebuah ekosistem platform web inovatif berbasis low-code architecture yang dirancang khusus untuk mentransformasikan, mengoptimalkan, dan mendigitalisasi proses pengawasan proyek konstruksi bangunan Gedung Negara Grahadi, Surabaya, Jawa Timur, Indonesia.
